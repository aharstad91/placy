# Placy WordPress Theme

## Theme Information
- **Theme Name:** Placy
- **Version:** 1.0.0
- **Author:** Andreas Harstad

## Required Files Created

This theme now includes all the minimum required files for WordPress theme approval:

1. **style.css** - Main stylesheet with theme header information
2. **functions.php** - Theme setup and functionality
3. **index.php** - Main template file
4. **header.php** - Header template
5. **footer.php** - Footer template
6. **sidebar.php** - Sidebar template
7. **screenshot.png** - Theme screenshot (needs to be created manually)

## Template Parts
- **template-parts/content.php** - Post content template
- **template-parts/content-none.php** - No content found template

## Features
- Custom logo support
- Navigation menus (Primary & Footer)
- Post thumbnails
- HTML5 support
- Widget areas
- Responsive design with Tailwind CSS
- Custom fonts (Adobe Typekit & Google Fonts)

## Installation
1. Upload the theme to `/wp-content/themes/placy/`
2. Activate the theme in WordPress Admin
3. Configure menus under Appearance > Menus
4. Add a custom logo under Appearance > Customize

## Notes
- The original `index.html` is preserved for reference
- JavaScript files are loaded from the `js/` directory
- Custom styles are loaded from `css/styles.css`
- Screenshot needs to be created (1200x900px recommended)
